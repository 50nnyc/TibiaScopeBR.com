# Publicacao no GitHub Pages

O site ja esta pronto como projeto estatico.

Arquivos que precisam ir para o repositorio `TibiaScopeBR.com`:

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`
- `README.md`

Depois de enviar para o GitHub:

1. Abra o repositorio no GitHub.
2. Va em `Settings` > `Pages`.
3. Em `Build and deployment`, selecione `Deploy from a branch`.
4. Escolha a branch `main` e a pasta `/root`.
5. Salve.

O endereco ficara parecido com:

`https://50nnyc.github.io/TibiaScopeBR.com/`

Para usar `TibiaScopeBR.com`, sera necessario comprar/controlar esse dominio e configurar DNS para GitHub Pages.
